VSCode Extension Yang Sering Digunakan Programmer Golang

  • #golang #tutorial #extension #vscode

Visual Studio Code memang sedang hits digandrungi oleh para programmer karena digadang lebih mudah penggunaannya dan lebih ringan jika dijalankan dalam suatu komputer. Selain mudah, vscode biasa para programmer sebut, banyak juga pendukung atau extension yang dapat menunjang kebutuhan programmer agar lebih mudah dalam melakukan pekerjaan. Extensin/plugin tambahan sering kita gunakan untuk membantu mempermudah programmer ketika sedang asik ngoding. Nah, berikut ini Santekno ada beberapa rekomendasi extension yang bisa digunakan agar pekerjaan kamu lebih mudah dan membantu lebih keren dalam melakukan ngoding.

GO (Extension Support for VS Code)

Package : https://marketplace.visualstudio.com/items?itemName=ms-vscode.Go

Secara resmi Microsoft mengeluarkan package untuk VSCode agar para programmer go bisa lebih leluasa dan menikmati melakukan program. Setelah kamu melakukan install, VSCode akan langsung memberikan pesan bahwa terdapat package yang saling terkait maka kamu tinggal tekan tombol install saja, secara otomatis nanti semua package kebutuhan untuk development go sudah bisa digunakan.

GitLens - Git Supercharged

Package: https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens

Bagi teman-teman yang biasa bekerja kolaborasi extension ini mempermudah jika ada perubahan kode yang kita tidak tahu kapan dan oleh siapa maka nanti perubahan kode tersebut dapat langsung dilihat agar bisa lebih memperhatikan kualitas codingan kita. Biasanya programmer bilang extension untuk ng-blame programmer lain jika codingan tersebut bermasalah atau terdapat bug.

Fitur-fitur yang disediakan

  • Annotation current line blame Anotasi informasi detail tentang author dengan cara kita hover ke codingan kita langsung terlihat detail informasinya.

  • Status Bar Blame memperlihatkan perstatus anotasi author dan waktu terakhir diubah.

  • Side Bar Views

  • Dan masih banyak lagi.

Bracket Pair Colorizer 2

Package: https://marketplace.visualstudio.com/items?itemName=CoenraadS.bracket-pair-colorizer-2

Extension ini sangat berguna banget bagi programmeer kalau lupa tanda kurung buka, apakah sudah ditutup atau belum. Nah dengan extension ini buka tutup kurung untuk codingan kita jadi lebih terlihat karena dilakukan pengecekan dan perubahan warna tiap buka kurung-nya sehingga lebih terlihat.

Better Comments

Package: https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments

Extension ini sangat berguna bagi kita agar lebih rapi untuk melakukan komentar dalam tiap function atau suatu variable jika ingin ditambahkan komentar. Extension ini mempermudah kita melihat komentar dalam suatu codingan agar mudah dibaca oleh programmer baru atau programmer awam ketika melihat codingan kita.
