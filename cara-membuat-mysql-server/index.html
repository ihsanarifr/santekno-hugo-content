Cara Membuat MySQL Server

  • #Linux #Sistem Operasi

Linux – Cara membuat MySQL Serverdi Fedora Core 7 itu lumayan gampang karena asalkan aplikasi mySQL pada Linux Anda sudah di instal maka bisa langsung di jalankan dengan baik dan bisa digunakan secar mudah. Tetapi kita harus tahu dulu perintah-perintah SQL agar nanti gampang untuk pembuatan tabel, database dan lain sebagainya.

/o:p
Aplikasi MySQL adalah aplikasi yang digunakan untuk membuat database semua sehingga bisa tercatat dan tersusun data dengan baik disana. Biasanya klo di windows kita sering menggunakan XAMPP dan disini juga kurang lebih seperti itu, jadi kita bisa mudah membuat database dalam Linux Fedora Core 7.
/o:p

Gambar MySQL Server
Tak sabar lagi untuk bermain dan mencoba. Kita langsung saja ke tahap-tahap awal membuat MySQL Server./o:p

A. Instalasi MySQL

Pengecekan pada Linux Anda sangat penting sekali karena agar bisa melihat apakah Linux kita sudah terpasang aplikasi MySQL server atau tidak. Jika tidak Instal saja MySQL server tersebut. Cara pengecekannya seperti berikut ini./o:p


rpm –qa | grep mysql/o:p

mysql-server-5.0.37-2.fc7/o:p
mysql-libs-5.0.37-2.fc7/o:p
mysql-5.0.37-2.fc7/o:p
php-mysql-5.2.2-3/o:p

Minimal yang diatas Aplikasi tersebut sudah diinstal agar bisa mendukung praktik kita mengenai MySQL Server ini. Kita juga memerlukan php-mysql untuk keperluan akses database MySQL via aplikasi web yang dibuat menggunakan bahasa PHP./o:p

B. Menggunakan MySQL

Setelah instalasi dilakukan, user sudah dapat mengakses server database menggunakan aplikasi client mysql. Pastikan service mysqld telah aktif./o:p

service mysqld restart/o:p

Kemudian melakukan koneksi ke server dengan perintah dibawah ini lalu [enter]/o:p

mysql/o:p

Maka akan keluar/o:p

Welcome to the MySQL Monitor. Commands end with ; or \g./o:p
Your MySQL connection id is 2/o:p
Server version: 5.0.37 Source distribution/o:p
Type ‘help;’ or ‘\h’ for help. Type ‘\c’ to clear the buffer./o:p
mysql>/o:p

Promt mysql> merupakan tanda bahwa kita sudah berhasil memasuki server. Pada prompt tersebut kita dapat mengetik perintah-perintah yang ditulis dengan bahasa SQL./o:p

Bahasa SQL (Structured Query Language) merupakan bahasa standar untuk server database. Dan kita akan sekilas membuat perintah-perintah SQL disini agar nanti bisa menggunakan MySQL Server di Linux ini dengan baik./o:p

Melihat versi MySQL/o:p

mysql> select version();/o:p

Melihat Tanggal/o:p

mysql> select current_date;/o:p

Melihat user/o:p

mysql> select user();/o:p

Melihat daftar database/o:p

mysql> show databases;/o:p

Memilih database, misalkan database bernama mysql/o:p

mysql> use mysql;/o:p

Menampilkan daftar tabel/o:p

mysql> show tables;/o:p

Menampilkan tipe data yang digunakan pada tabel user/o:p

mysql> describe user;/o:p

Menampilkan isi field user dan host dari tabel user/o:p

mysql> select user, host from user;/o:p

Membuat database bernama dbsantekno/o:p

mysql> create database dbsantekno;/o:p

Membuat tabel peserta pada database dbsantekno. Tabel peserta terdiri atas dua buah field, yaitu Nama dan Alamat/o:p

mysql> use dbsantekno;/o:p
mysql> create table peserta (Nama char(20), Alamat char(50));/o:p

Memasukkan record pada tabel peserta. Kita akan memasukkan dua buah record saja/o:p

mysql> insert into peserta values(‘Ihsan Arif’, ‘Tasikmalaya’);/o:p

mysql> insert into peserta values(‘Arif Rahman’,’Tasikmalaya’);/o:p

Menampilkan isi tabel peserta/o:p

mysql> select * from peserta;/o:p
+——————–+—————–+/o:p
|  Nama            |  Alamat        |/o:p
+——————+—————-+/o:p
|  Ihsan Arif      |  Tasikmalaya   |/o:p
|  Arif Rahman     |  Tasikmalaya   |/o:p
+——————+—————-+/o:p
2 rows in set (0.00 sec)      /o:p

Keluar dari server MySQL/o:p

mysql> exit/o:p


C. Konfigurasi MySQL

Mengenai keamanan itu merupakan hal yang penting bagi sebuah server database. Apalagi jika data tersebut sangat penting dan tidak boleh di akses oleh setiap user. Secar Default setelah MySQL diinstal maka setiap user diizinkan mengakses semua data server menggunakan login root tanpa harus mengetikkan password. Maka harus menentukan password root agar tidak bisa di akses oleh sembarang user. Gunakanlah utilitas mysqladmin untuk menentukan password root./o:p


Sekian dulu untuk penjelasan mengenai Cara membuat MySQL Server, biar lebih paham lagi mengenai pembuatan MySQL Server anda harus bisa perbanyak latihan mengenai bahasa SQLnya. Semoga Bermanfaat./o:p

mysqladmin password “root1234”/o:p

Maka jika akan login ke MySQL Server perintahnya harus seperti ini/o:p

mysql –u root –p/o:p

Maka akan keluar perintah untuk memasukkan password dan passwordnya yang telah dibuat tadi yaitu “root1234”. Nah, sekarang user tidak bisa mengakses server dengan perintah mysql saja./o:p

Jika ingin mengubah password root tinggal perintah seperti dibawaih ini/o:p

mysqladmin password “rootpass” –p/o:p

Untuk menambah user dan menentukan hak aksesnnya dapat ditentukan dengan dua cara yaitu:/o:p

Cara pertama menggunakan perintah GRANT, polanya :/o:p

mysql> GRANT …/o:p
ON ./o:p
TO myusername@”%.mydomainname.com”/o:p
IDENTIFIED BY ‘mypassword’;/o:p

Cara kedua menggunakan perintah INSERT, polanya :/o:p

mysql> insert mysql;/o:p
mysql> INSERT INTO user VALUES (‘%mydomainname.com’, ‘myusername’, PASSWORD(‘mypassword’),…);/o:p
mysql> FLUSH PRIVILEGES;/o:p

Untuk yang kedua tidak disarankan karen ini terlalu panjang seperti halnya untuk membuat record dalam sebuah tabel./o:p

Misalkan kita akan membuat user bernama santekno dengan password santekno1234. User ini dizinkan mengakses database dari komputer santekno.com. Login terlebih dahulu selanjutnya ketikkan perintah sebagai berikut :/o:p

mysql> GRANT all privileges/o:p
ON . TO santekno@”santekno.com”/o:p
IDENTIFIED BY ‘santekno1234’;/o:p

Demikian tutorial tentang CaraMembuat MySQL. Semoga bisa membantu semua teman-teman dalam menjalankan semua hal berhubungan dengan jaringan ini./o:p
