Perintah Dasar Linux Part 3

  • #Linux #Sistem Operasi

Setelah Anda Mempelajari pada postingan sebelumnya Perintah Dasar Part 2. Kini tinggal melanjutkan tahapan kita, agar bisa lebih mahir dan bertambah pengetahuan juga, kita melanjutkan postingan Kali ini dengan Perintah Dasar Linux.

Manajemen Akses pada file

Pada postingan kali ini, akan menjelaskan suatu file itu bisa di akses oleh beberapa orang user, owner, other ataupun grup. Biasa sering kita sebut dengan Mode Akses dan Owner. Setiap file (termasuk) dapat diatur hak aksesnya menggunakan perintah chmod. Inilah yang disebut mode akses (atau permission). Mode akses dapat diubah oleh root atau owner (pemilik) file tersebut. Untuk mengetahui informasi owner, maka akses dan informasi lain gunakan saja perintah ls –l. Sebagai contoh, untuk menampilkan owner dan mode akses file a.txt yang ada di direktori /home/santekno gunakan perintah :

ls –l   /home/santekno/a.txt

Contoh output perintah diatas sebagai berikut :

-rw-r--r-- 1 santekno santekno   0  Sep 5 07:05 a.txt

Pola penulisan chmod ada dua, yaitu : Secara simbolik :

chmod [who] [OPTION] [perm] [ARGUMENTS]

Secara numerik :

chmod nnn [ARGUMENTS]

Keterangan

  • [who] : u atau g atau o atau
  • [OPTION] : + atau –
  • [perm] : r atau w atau x
  • nnn : kombinasi tiga angka dari 000 s.d. 777

Mode Akses

Mode akses digunakan untuk mengetahui tiga kategori user, * User pemilik (owner) file : u * User yang satu group dengan owner : g * User yang berbeda group dengan owner : o

Jenis Mode Akses

Hak baca : r atau 4 Hak tulis : w atau 2 Hak eksekusi : x atau 1

Simbol “-“ artinya menonaktifkan suatu hak akses.

Contoh pemakaian perintah chmod :

chmod ug-w /tmp/ok.txt
chmod ug-w,o+x /tmp/ok.txt
chmod 644 /tmp/ok.txt
chmod uo=r /tmp/ok.txt
chmod –R ugo+rwx /home/santekno

Sangatlah penting untuk memahami mode akses dan chmod. Jika belum paham sebaiknya memperbanyak latihan menggunakan chmod. Umumnya admin Linux itu lebih menyukai cara numerik dibandingkan simbolik.

Admin dapat mengubah mode akses dan owner file milik semua user. Sedangkan user biasa hanya boleh mengubah mode akses file miliknya sendiri, dan tidak bisa mengubah owner file miliknya. Hanya admin yang berhak mengubah owner file semua user.

Admin dapat mengubah owner file menggunakan perintah chown. Sebagai contoh, untuk mengubah owner file menjadi arif, perintahnya sebagai berikut :

chown arif a.txt

Untuk mengubah owner dan group file :

chown arif.arif a.txt

Untuk mengubah group file menjadi group santekno :

chown :santekno a.txt
