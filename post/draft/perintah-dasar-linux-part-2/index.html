Salah satu konseip yang elegan dari Linux adalah “everything is a file”. Dimana segala sesuatu ada di Linux, mulai dari hardware, software direktori, dan sebagainya diperlakukan sebagai file. Oleh sebab itu, sangatlah penting memahami filesystem Linux. Setidak-tidaknya dari sisi praktis. Beberapa perintah penting yang terkait dengan manajemen file dapat dilihat di tabel berikut.

Kita dapat membedakan jenis file dan informasi lain menggunakan perintah ls –l. Perhatika contoh berikut :/o:p

ls –l   /etc/o:p

Contoh output perintahnya sebagai berikut :/o:p

drwxr-xr-x  2 root root 4096 Sep 15 2006 4Suite/o:p
-rw-r–r–  2 root root 15289 Mar 23 2005 passwd/o:p
