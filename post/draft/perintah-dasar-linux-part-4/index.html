Perintah Dasar Linux Part 4

  • #Linux #Sistem Operasi

Nah, masih cukup banyak perintah-perintah Linux yang dapat Anda Pelajari. Namun, apa yang di cantumkan pada tabel merupakan perintah-perintah inti. Selanjutnya akan di bahas utilitas manajemen aplikasi di lingkungan Fedora. Tambahan untuk melihat versi aplikasi dan menampilkan beberapa daftar aplikasi yang sudah termasuk di Linux Anda bisa digunakan sebagai berikut./o:p

a.       Menampilkan daftar aplikasi/o:p
          daftar aplikasi yang sudah diinstal dapat dilihat. Gunakan perintah :/o:p

rpm -qa/o:p

          Jika terlalu banyak, filter saja tampilannya menggunakan more./o:p

rpm –qa | more/o:p

b.      Menampilkan daftar file suatu aplikasi/o:p
          Daftar file-file yang diinstal oleh suatu aplikasi dapat ditampilkan. Misalnya saja ada aplikasi bernama nmap-4.20-5.fc7. Gunakan perintah/o:p

rpm –ql nmap-4.20-5.fc7/o:p

          Atau cukup dengan perintah/o:p
          /o:p
rpm –ql nmap/o:p

c.       Menampilkan informasi Suatu aplikasi/o:p
          Untuk mengetahui informasi suatu aplikasi, misalkan aplikasi bernama nmap-4.20-5.fc7. gunakan perintah :/o:p

rpm –qi nmap-4.20-5.fc7/o:p

d.      Menampilkan daftar file aplikasi yang belum diinstal/o:p
          untuk mengetahui daftar file suatu aplikasi yang belum diinstal misalnya file-nya bernama webmin-1-290-1.noarch.rpm. Gunakan perintah :/o:p

rpm –qlp webmin-1-290-1.noarch.rpm/o:p

e.       Menginstal aplikasi/o:p
          Instalasi file dapat dilakukan dengan mudah sepanjang semua kebutuhan sudah tersedia. Misalkan file-nya bernama webmin-1-290-1.noarch.rpm. Gunakan perintah :/o:p
          /o:p
rpm –ivh webmin-1-290-1.noarch.rpm/o:p

f.       Meng-unistal aplikasi/o:p
          aplikasi yang sudah diinstal dapat di-remove. Misalkan aplikasi bernama webmin-1-290-1.noarch.rpm. Gunakan perintah :/o:p

rpm –e webmin-1-290-1.noarch.rpm/o:p

Nah, sejauh ini kita sudah melihat beberapa perintah Linux yang penting. Pahamilah semua perintah tersebut agar memperkuat pemahaman terhadap perintah Linux./o:p
