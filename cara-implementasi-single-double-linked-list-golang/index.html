Cara Implementasi Single Double Linked List Golang

  • #golang #tutorial #single linked list #double linked list

Pengertian Dasar

Jika kamu sudah pernah baca-baca soal Linked List bisa baca terlebih dahulu postingan Santekno sebelumnya single linked list dan double linked list

Berbeda dengan sebelumnya implementasinya menggunakan bahasa C++, sekarang kita menggunakan bahasa GO alias golang biasanya orang bilang. Sebetulnya tidak terlalu jauh dan ribet untuk mengaplikasikannya karena script dari golang lebih mudah dan ringkas.

Single Linked List

Single Linked List kalau kita simpulkan hanya memiliki Next() Node saja. Ini adalah contoh simpel dari single linked list.

Double Linked List

Apa yang membedakan dari single dan double linked list? Yaitu di dalam Double Linked List node itu bisa tau sebelum dan setelah Node yang sedang dituju. sehingga kita tambahkan method Prev() agar node sebelumnya itu kita tahu alamat memory-nya.

Library Go

Jika Anda ingin mempelajari lebih lanjut terkait Linked List bisa coba menggunakan library Go yang sudah disediakan disini. Ini merupakan library yang bisa mempermudah Anda untuk mengimplementasikan single linked list ataupun double linked list.
